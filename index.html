O que são as 'Ferramentas de anonimização' e como usá-las?
>

Nossas ferramentas de anonimização' permitem que você desfoque, corte ou mascare qualquer foto que você enviar para o nosso site. Toda vez que você enviar uma foto, você será levado ao nosso assistente fotográfico para editá-la ao seu gosto. Você pode editar as suas fotos para ficarem tão discretas e não identificáveis quanto você desejar.

Como funcionam as 'Chaves Privadas'?
>

Se você tiver enviado fotos privadas, outros usuários terão que solicitar acesso antes que eles possam visualizá-las. Isso é chamado de pedido de chave privada. (Por favor, note que as chaves privadas precisam ser solicitadas por um usuário, antes que possam ser aprovadas. Não é possível conceder ao usuário acesso às suas fotos privadas se eles não tiverem solicitado.)

Para aprovar uma chave privada para outro usuário:

- Você verá uma notificação quando receber uma nova solicitação

- Vá para a aba 'Chaves Privadas' no lado esquerdo na lista de menus.

- Agora você verá uma lista de todas as solicitações de chave pendentes

- Clique no botão azul 'Permitir' se desejar aprovar uma chave ou no branco 'Negar' se quiser rejeitar o pedido. Se você aprovar a solicitação, o usuário poderá visualizar todas as suas fotos privadas.

Para solicitar uma chave privada para outro usuário:

- Vá para o perfil do usuário

- Clique no botão 'Solicitar uma Foto'

- O usuário receberá uma notificação de sua solicitação e aprovará ou rejeitará o seu pedido. Uma vez aprovado, você poderá visualizar as fotos privadas do usuário.

    • Meu pagamento é seguro?
    >

    A segurança é a prioridade número #1 em Sasha7. Sasha7 cumpre à risca o GDPR e segue as regulamentações mais rigorosas possíveis. Usamos criptografia de 256 bits, os serviços web mais confiáveis e SLA para manter nossos usuários seguros!
